# Research Corpus

## Overview

This directory contains technical research organized by topic. Each topic folder contains a comprehensive analysis with the top 10 authoritative sources, license information, and applicability assessments.

## Research Methodology

For every technical subject, at least the top 10 authoritative sources are researched, documented, and cross-referenced. Each research document includes:

- Sources with URLs
- License information
- Summary
- Strengths and weaknesses
- Applicability to the platform
