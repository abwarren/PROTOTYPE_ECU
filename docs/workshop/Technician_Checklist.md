# ✅ Technician Checklist

> **Audience:** Workshop technicians
> **Purpose:** Standardized checklist for every ECU installation and tuning job.

---

## Pre-Installation

- [ ] Customer vehicle details recorded
- [ ] ECU compatibility verified
- [ ] All required parts in stock (ECU, harness, sensors, etc.)
- [ ] Battery fully charged (>12.5V)
- [ ] Tools ready: multimeter, timing light, laptop with Studio, wideband O2

## Installation

- [ ] Battery disconnected
- [ ] ECU mounting location verified (cabin recommended)
- [ ] Wiring diagram reviewed
- [ ] Power connections: battery +12V, ground, ignition switched
- [ ] All fuses installed at rated values
- [ ] Sensor wiring verified (MAP, TPS, CLT, IAT, O2, knock)
- [ ] Output wiring verified (injectors, ignition, fuel pump, fans)
- [ ] CAN bus wiring verified (if used)
- [ ] All connections secured and protected
- [ ] Battery reconnected
- [ ] Power-on test: ECU LED on, no smoke, no smells

## Base Configuration

- [ ] Engine type and displacement set
- [ ] Trigger pattern configured and verified
- [ ] Injector flow rate and dead time set
- [ ] Ignition type configured (COP, wasted spark, distributor)
- [ ] Base timing verified with timing light (±1°)
- [ ] Fuel pressure verified
- [ ] Base VE and timing tables loaded

## Startup

- [ ] Cranking fuel set (prime pulse)
- [ ] Engine starts and idles
- [ ] No unusual noises or vibrations
- [ ] Coolant temperature rising normally
- [ ] Oil pressure within spec
- [ ] No coolant or oil leaks

## Tuning

- [ ] Idle AFR at target (±0.1)
- [ ] Idle stable (RPM fluctuation < 50 RPM)
- [ ] Part-throttle drivability smooth
- [ ] Tip-in and tip-out response good
- [ ] Full-throttle AFR safe (12.0-12.8:1)
- [ ] No knock detected
- [ ] Timing curve verified against table

## Validation

- [ ] Road test completed with datalogging
- [ ] All systems verified (cooling fans, fuel pump, gauges)
- [ ] No DTCs present
- [ ] Customer test drive (optional)
- [ ] Final datalog saved to cloud/workshop records

## Handover

- [ ] Mobile app installed on customer phone (if cloud-enabled)
- [ ] Quick reference card provided
- [ ] 1-week follow-up scheduled
- [ ] 1-month review scheduled
- [ ] Invoice provided
