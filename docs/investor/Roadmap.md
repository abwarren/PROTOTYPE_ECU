# Product Roadmap

> **Living document** — Updated with every milestone.

---

## Phase 1: MVP Foundation ✅ Complete

**Goal:** Working ECU platform with documented architecture and proven firmware foundation.

- Repository structured (white-label, ready for any brand)
- rusEFI firmware fork established (proven on thousands of engines)
- Architecture fully documented
- Engineering processes established (DDD policy)
- Brand abstraction layer (rebrand without code changes)

**Completion:** 100%

---

## Phase 3: Cloud Platform (Q1-Q2 2027)

**Goal:** Connected ECUs with remote diagnostics, OTA updates, and fleet management.

- Cloud API and authentication
- Telemetry ingestion and storage
- OTA firmware update infrastructure
- Mobile app (BLE diagnostics)
- User account and vehicle profile management
- Fleet dashboard for workshops

**Completion:** ~5%

---

## Phase 4: AI Platform (Q3-Q4 2027)

**Goal:** AI-assisted tuning that makes tuners faster and more consistent.

- AI calibration suggestions
- Automated fuel map optimization
- Predictive diagnostics
- Natural language query interface

**Completion:** 0%

## Phase 5: Fleet Platform (2028)
