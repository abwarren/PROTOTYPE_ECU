# Features

## What the Platform Delivers (Value Language)

### Prototype ECU Hardware

| Feature | What It Means for Customers |
|---------|-----------------------------|
| Professional performance | Controls any 4-12 cylinder engine with precision |
| Wide engine support | Gasoline, flex-fuel, boosted, naturally aspirated, rotary |
| Modern connectivity | USB-C, dual CAN bus, BLE, optional WiFi/cellular |
| Professional grade | IP67 sealed, Deutsch connectors, -40°C to +125°C |
| Affordable | 1/3 to 1/2 the price of competitor ECUs |

### Prototype Studio (Desktop App)

| Feature | What It Means for Customers |
|---------|-----------------------------|
| Modern dashboard | Real-time gauges that look and feel premium |
| Drag-and-drop layout | Customize your workspace in seconds |
| Dark mode | Easy on the eyes during late-night tuning sessions |
| Search everywhere | Find any setting instantly — no more menu hunting |
| Global palette | One keyboard shortcut to search, navigate, and execute |
| Dockable windows | Arrange tools exactly how you want them |
| 3D table editors | Edit fuel and ignition maps with intuitive controls |
| Auto-tuning | Closed-loop wideband automatically adjusts fuel tables |
| Datalogging | Record, playback, and analyze engine data at 1ms resolution |
| Cloud backup | Your calibrations are never lost |
| Plugin system | Extend functionality without modifying the core |
| AI panel | Intelligent suggestions (starting with rule-based, evolving to ML) |
